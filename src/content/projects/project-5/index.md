A data platform to inform citizens about incumbent representatives. Architected the core MVP enabling users to explore 38 Texas Congressional districts via interactive maps, candidate profiles, and legislative tracking across 100+ policy-categorized bills. Built a RAG-based ETL pipeline to embed and index congressional bill data in ChromaDB for fast semantic search and real-time query capabilities.
