Built a production journaling mobile app that integrates a custom fine-tuned BERT sentiment analysis model for real-time emotion detection. The model was trained and published to Hugging Face Hub, then served via a FastAPI backend. Users receive personalized insights based on their journal entries' emotional patterns, with data persisted in Supabase. The app is built with React Native and Expo for cross-platform iOS and Android support.
